# Contributing to Payroll Pro

Thank you for your interest in contributing to Payroll Pro!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Install dependencies: `npm install`
4. Set up environment variables (see SETUP.md)
5. Create a feature branch: `git checkout -b feature/your-feature-name`

## Code Style

- Use TypeScript for all new code
- Follow existing code patterns
- Run `npm run lint` before committing
- Run `npm run format` to format code

## Commit Messages

Use clear, descriptive commit messages:
- `feat: add employee search functionality`
- `fix: correct payroll calculation for overtime`
- `docs: update setup instructions`

## Pull Request Process

1. Ensure all tests pass
2. Update documentation if needed
3. Add/update tests for new features
4. Submit PR with clear description
5. Address review feedback

## Project Structure

- `apps/web` - Next.js web application
- `apps/mobile` - Expo React Native app
- `libs/ui` - Shared UI components
- `libs/utils` - Utility functions
- `libs/data` - Database models and Prisma schema

## Questions?

Open an issue for questions or discussions.
